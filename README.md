# Ultimate Tic-Tac-Toe Web Project

A complete web implementation of Ultimate Tic-Tac-Toe with Player vs Player and Player vs AI modes, featuring an attractive interface with dark/light theme, customizable settings, and AI with multiple difficulty levels.

## Features

### Game Modes

- **Player vs Player**: Play locally with a friend on the same device
- **Player vs AI**: Challenge an intelligent AI agent with three difficulty levels

### AI Difficulty Levels

1. **Easy**: Makes mostly random moves with basic logic
2. **Medium**: Uses Minimax algorithm with limited depth search
3. **Hard**: Uses Minimax algorithm with Alpha-Beta pruning for optimal play

### Customization Options

- Dark/Light theme toggle
- Customizable player names
- Multiple theme color options
- Game settings:
  - Highlight valid moves
  - Show move hints
  - Animation effects
  - Sound effects

### Game Features

- Interactive board with visual feedback
- Move history tracking
- Game timer and move counter
- Visual indicators for:
  - Current player's turn
  - Next required board
  - Won small boards
  - Game status
